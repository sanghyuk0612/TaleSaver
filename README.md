# 로그라이크 게임 졸업작품

## 프로젝트 개요
본 레포지토리는 졸업작품으로 제작한 로그라이크 게임 프로젝트입니다. 총 5명이 팀으로 참여하여 기획 및 개발을 진행했습니다.

이 게임의 핵심 특징은 **매 플레이마다 달라지는 맵 구조**로, 반복 플레이 시에도 매번 다른 전략과 진행 경험을 제공하는 데 초점을 두고 있습니다.

## 장르
- Roguelike

## 기본 스토리
게임에는 기본적인 세계관과 스토리가 존재하며, 플레이어는 각 챕터를 진행하면서 전투와 선택을 반복해 최종 보스에 도달하게 됩니다.

※ 스토리의 서사 구조 및 엔딩 분기 여부는 추가 설명이 필요할 수 있습니다.

## 게임 구조
- 총 3개 챕터로 구성
- 각 챕터 구성
  - 전투 스테이지: 8개
  - 상점 스테이지: 2개
  - 보스 스테이지: 1개

## 전투 및 성장 시스템
- 몬스터를 사냥하면 아이템 드롭
- 획득한 아이템은 상점에서 거래 가능
- 아이템 가격은 **상점 라운드마다 변동**됨

※ 아이템 희귀도, 강화 요소, 영구 성장 여부는 명확하지 않을 수 있음

## 맵 시스템
- 여러 종류의 맵 존재
- 매 판 맵이 무작위로 구성되어 동일한 플레이 흐름이 반복되지 않도록 설계

## 보스
- 총 3종의 보스 존재
- 최종 보스는 **2페이즈 전투**로 구성

## 캐릭터
- 현재 플레이 가능 캐릭터: 1종
- 추가 캐릭터는 기획 단계에 존재
- 모든 캐릭터는 스킬 4개 보유를 전제로 설계됨
